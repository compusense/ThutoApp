# **App Name**: CampusConnect

## Core Features:

- SuperAdmin Login: Secure login for the Regional Office Administrator to manage central and regional data.
- Teacher Login: Secure login for teachers to enter exam and assessment marks.
- School Head Login: Secure login for School Administrators to manage school data and teacher allocations.
- User Role Management: Database of user profiles stored in Firestore.
- Firestore Integration: Utilize Firestore to store and manage school, teacher, and student data efficiently.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and authority.
- Background color: Light blue (#E8EAF6) for a calm and professional backdrop.
- Accent color: Vibrant orange (#FF9800) to highlight key actions and notifications.
- Body text: 'PT Sans' for a modern and readable interface.
- Headline font: 'Space Grotesk', sans-serif, for headlines and short amounts of body text; if longer text is anticipated, use this for headlines and 'Inter' for body.
- Use clear and consistent icons to represent different modules and actions.
- Implement a responsive layout that adapts to different screen sizes.
